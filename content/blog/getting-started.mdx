# Welcome to Your HUD Portfolio

This is a sample blog post to demonstrate the MDX blog system. You can write posts using Markdown with JSX support.

## How to Create a Post

1. Create a new `.mdx` file in the `content/blog/` directory
2. Add frontmatter at the top (title, date, excerpt, tags)
3. Write your content in Markdown
4. Push to deploy — your post will appear automatically

## Markdown Features

You can use all standard Markdown features:

- **Bold text** and *italic text*
- [Links](https://example.com)
- Code blocks with syntax highlighting
- Lists, headings, blockquotes, and more
